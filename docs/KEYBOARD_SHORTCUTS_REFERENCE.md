# Keyboard Shortcuts Reference
**Complete list of all keyboard shortcuts in VectorForge**

---

## UI Automation Shortcuts

| Shortcut | Action | Description |
|----------|--------|-------------|
| `Ctrl+K` / `Cmd+K` | Open Keyboard Shortcuts | View all shortcuts |
| `Ctrl+Shift+P` / `Cmd+Shift+P` | Project Wizard | Create new project |
| `Ctrl+Shift+T` / `Cmd+Shift+T` | Template Library | Browse code templates |

---

## File Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+N` | New File |
| `Ctrl+O` | Open File |
| `Ctrl+S` | Save |
| `Ctrl+Shift+S` | Save As |

---

## Edit Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+Z` | Undo |
| `Ctrl+Shift+Z` | Redo |
| `Ctrl+X` | Cut |
| `Ctrl+C` | Copy |
| `Ctrl+V` | Paste |
| `Ctrl+A` | Select All |

---

## View Operations

| Shortcut | Action |
|----------|--------|
| `Ctrl++` | Zoom In |
| `Ctrl+-` | Zoom Out |
| `Ctrl+0` | Fit to Screen |
| `Ctrl+1` | Actual Size (100%) |

## Object Operations

---

## Tips

- **Practice Mode:** Open Keyboard Shortcuts panel (`Ctrl+K`) and toggle Practice Mode to highlight keys as you press them
- **Customization:** Keyboard shortcuts can be customized (coming soon)
- **Search:** In Keyboard Shortcuts panel, search for any shortcut by name or key combination

---

**All shortcuts work with both `Ctrl` (Windows/Linux) and `Cmd` (Mac)!**
