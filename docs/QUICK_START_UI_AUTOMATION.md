## Step 2: Try Project Wizard

1. Click **"Set Up Project"** in Action Center
2. Click **"Next"** on the welcome screen
3. Enter a project name (e.g., "my-awesome-project")
4. Choose a project type (React, Node.js, etc.)
5. Click **"Next"** through the steps
6. Review your settings
7. Click **"Create Project"**

**That's it!** No terminal, no commands - just click through the wizard.

---

## Step 3: Browse Templates

1. Click **"Browse Templates"** in Action Center
2. See 5 example templates:
   - Service Base (TypeScript service class)
   - React Component (with error boundary)
   - API Route (Express.js)
   - Type Definition (TypeScript interface)
   - Test Suite (Jest)
3. Click a template to see the code preview
4. Click **"Use This Template"** to generate code

---

## Step 5: Learn Keyboard Shortcuts

1. Click **"Keyboard Shortcuts"** in Action Center (or press `Ctrl+K`)
2. Search for "save" or "undo"
3. Toggle **"Practice Mode"** to highlight keys as you press them
4. Learn shortcuts by category (File, Edit, View, Tools)
