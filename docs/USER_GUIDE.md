# VectorForge User Guide
**For Everyone - Especially Beginners**

## Welcome to VectorForge!

VectorForge is a **team-based vector graphics editor** - a tool for creating digital art, illustrations, and animations, designed for both solo creators and teams. If you've never used software like Adobe Illustrator or Inkscape, don't worry! This guide will help you get started.

**Team Features:** VectorForge includes built-in task management, sprint planning, and collaboration tools. Whether you're working alone or with a team, these features help you stay organized and productive.

## Basic Concepts

### Layers

**Layers** are like transparent sheets stacked on top of each other. Each layer can contain shapes, and you can:
- Reorder layers (move up/down)
- Show/hide layers
- Lock layers (prevent editing)
- Group layers together

**Why use layers?**
- Organize your artwork
- Edit parts separately
- Control what's visible

### Tools

**Selection Tool** (`V`)
- Click objects to select them
- Move selected objects
- Resize with handles

**Rectangle Tool** (`M`)
- Draw rectangles and squares
- Hold `Shift` for perfect squares

**Ellipse Tool** (`L`)
- Draw circles and ovals
- Hold `Shift` for perfect circles

**Pen Tool** (`P`)
- Draw custom paths
- Click to place points
- Create curves and lines

**Text Tool** (`T`)
- Add text to your artwork
- Click canvas to place text

### Properties

When you select an object, you can change:
- **Fill Color** - The inside color
- **Stroke Color** - The outline color
- **Stroke Width** - How thick the outline is
- **Opacity** - How transparent it is

## Common Tasks

### Creating a Simple Logo

1. Draw a shape (rectangle, circle, etc.)
2. Change the fill color
3. Add text with the Text tool
4. Arrange elements
5. Group layers together
6. Export as SVG or PNG

### Making an Animation

1. Create your artwork
2. Open the Timeline (bottom panel)
3. Add a keyframe at frame 0
4. Move to frame 10
5. Move or change your object
6. Add another keyframe
7. Press Play to see animation

### Saving Your Work

**Save Project** (`Ctrl+S`)
- Saves as `.xibalba` file
- Keeps all layers and properties
- Can reopen later

**Export as SVG**
- For web use
- Scalable vector format
- Edit in other programs

**Export as PNG** (coming soon)
- For images
- Fixed size
- Good for sharing

## Troubleshooting

### The App Won't Load
- Check your internet connection
- Refresh the page (`F5`)
- Clear browser cache
- Check Error Dashboard

### Tools Don't Work
- Make sure you're on the canvas (white area)
- Check if a layer is locked
- Try selecting the tool again
- Check Error Dashboard for errors

### Can't See My Drawing
- Check layer visibility (eye icon)
- Zoom in/out (`Ctrl++` / `Ctrl+-`)
- Reset view (`Ctrl+0`)
- Check if object is outside canvas

### Something Broke
- Press `Ctrl+Z` to undo
- Check Error Dashboard
- Save your work
- Refresh the page (you'll lose unsaved work)
